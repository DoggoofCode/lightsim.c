# Light Sim

This is a simple lightsim based on a cellular automata, using raylib, by perterbing the cells next to it in order to create a wave. 

## Running
> This project is still a WIP. Feel free to change the settings in main.c until I get another system working

To run this project run:
```bash
[git clone https://github.com/DoggoofCode/lightsim.c.git](https://github.com/DoggoofCode/lightsim.c.git)

make && ./lightsim
```
